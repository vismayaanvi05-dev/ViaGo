🚀 HyperServe Customer App

Choose how you want to test the mobile app

📱 Option 1: Test on Phone (Best Experience)

  • Install Expo Go app from App Store/Play Store
  • Run this command in terminal:
cd /app/mobile-app && yarn start
  • Scan QR code with Expo Go app
  • App loads in 30 seconds

🌐 Option 2: Test in Browser (Quick Preview)

Run the app in web browser (mobile-like experience):

cd /app/mobile-app && yarn web

Then open: http://localhost:19006

Open Web Preview

🖥️ Option 3: Direct Terminal Access

If you have terminal access, run:

bash /app/mobile-app/launch.sh

This will start Expo and show you the QR code in terminal.

📖 Need Help?

Check the detailed guides:

  • /app/mobile-app/TESTING_GUIDE.md
  • /app/mobile-app/README.md

Note: The QR code is generated when you run yarn start

It cannot be pre-generated as it contains your local network address.
